# 📊 SQL Exercise 1 — Pixar Movies

This project demonstrates basic SQL queries using the `movies` table.

---

## 🗂️ Table: Movies

Contains data about Pixar films including title, director, year, and duration.

---

## ✅ 1. Find the title of each film

```sql
SELECT title FROM movies;
```

![Title Output](screenshots/title_query.png)

---

## ✅ 2. Find the director of each film

```sql
SELECT director FROM movies;
```

![Director Output](screenshots/director_query.png)

---

## ✅ 3. Find the title and director of each film

```sql
SELECT title, director FROM movies;
```

![Title and Director](screenshots/title_director.png)

---

## ✅ 4. Find the title and year of each film

```sql
SELECT title, year FROM movies;
```

![Title and Year](screenshots/title_year.png)

---

## ✅ 5. Find all the information about each film

```sql
SELECT * FROM movies;
```

![All Data](screenshots/table_view.png)

---

## 🚀 Learning Outcome

- Learned how to use `SELECT` statements  
- Retrieved single and multiple columns  
- Used `*` to fetch complete table data
